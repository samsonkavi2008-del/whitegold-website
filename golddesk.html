⚙️ Supabase Setup Required — One Time Only

To activate the live login system, follow these 4 steps:

Step 1 — Go to supabase.com → Sign up free → Create a new project called golddesk

Step 2 — In Supabase dashboard, go to SQL Editor and run this SQL:

create table staff (
  id uuid default gen_random_uuid() primary key,
  employee_id text unique not null,
  full_name text not null,
  email text unique not null,
  role text check (role in ('admin','manager')) not null,
  branch text not null,
  created_at timestamptz default now()
);

-- Insert first admin
insert into staff (employee_id, full_name, email, role, branch)
values ('WG-ADMIN', 'Admin Owner', 'admin@whitegold.in', 'admin', 'Head Office');

Step 3 — Go to Authentication → Users → Click Invite User → Enter admin@whitegold.in → Set password

Step 4 — Go to Project Settings → API → Copy your Project URL and anon public key → Paste them below in the golddesk.html file where it says YOUR_SUPABASE_URL and YOUR_SUPABASE_ANON_KEY

In this file, find and replace:

const SUPABASE_URL = 'YOUR_SUPABASE_URL'

const SUPABASE_KEY = 'YOUR_SUPABASE_ANON_KEY'

✅ Once configured, this page will show the login form automatically.
